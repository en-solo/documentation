## Accessibility

### Contrast Ratios

All color combinations must meet WCAG 2.1 AA standards at minimum:

<CardGroup cols={2}>
  <Card title="Normal Text" icon="text">
    **Minimum ratio: 4.5:1**

    For text smaller than 18px (or 14px bold)
  </Card>

  <Card title="Large Text" icon="heading">
    **Minimum ratio: 3:1**

    For text 18px+ (or 14px+ bold)
  </Card>

  <Card title="UI Components" icon="square-check">
    **Minimum ratio: 3:1**

    For interactive elements and graphics
  </Card>

  <Card title="AAA Standard" icon="medal">
    **Enhanced ratios**

    Normal text: 7:1, Large text: 4.5:1
  </Card>
</CardGroup>

### Color Blindness

<Warning>
  Never rely solely on color to convey information. Always pair color with text, icons, or patterns.
</Warning>

Test all color combinations for:
- Protanopia (red-blind)
- Deuteranopia (green-blind)
- Tritanopia (blue-blind)
- Achromatopsia (total color blindness)

## Usage Guidelines

<Steps>
  <Step title="Start with semantic meaning">
    Choose colors based on their semantic meaning (primary for main actions, error for warnings, etc.)
  </Step>

  <Step title="Check contrast">
    Verify contrast ratios meet accessibility standards using tools like WebAIM Contrast Checker
  </Step>

  <Step title="Use consistently">
    Apply the same color for the same purpose across all interfaces
  </Step>

  <Step title="Limit your palette">
    Use 2-3 colors per component or section to avoid visual clutter
  </Step>
</Steps>

## Dark Mode

<Info>
  When implementing dark mode, use darker variants for backgrounds and lighter variants for text and interactive elements. Maintain the same contrast ratios as light mode.
</Info>

| Element | Light Mode | Dark Mode |
|---------|------------|-----------|
| Background | Gray 50 (`#fafafa`) | Gray 900 (`#18181b`) |
| Surface | White (`#ffffff`) | Gray 800 (`#27272a`) |
| Text Primary | Gray 900 (`#18181b`) | Gray 50 (`#fafafa`) |
| Text Secondary | Gray 600 (`#52525c`) | Gray 400 (`#9f9fa9`) |
| Borders | Gray 200 (`#e4e4e7`) | Gray 600 (`#52525c`) |
