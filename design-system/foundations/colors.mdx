## Overview

The Kopi color system uses a minimal, sophisticated palette centered around blacks, grays, and subtle brand accents. The system prioritizes readability and creates a clean, modern aesthetic.

## Color Palette

### Primary Colors

Primary colors are used for text, buttons, and key UI elements.

<AccordionGroup>

    **Primary Black**

    - **Hex**: `#17171a`
    - **RGB**: `rgb(23, 23, 26)`
    - **Usage**: Primary text, primary button backgrounds, main UI elements

    ```css

    ```

  <Accordion title="Dark Gray" icon="circle">

    - **RGB**: `rgb(38, 38, 42)`

    ```css

    ```
  </Accordion>

  <Accordion title="Gray" icon="circle">

## Usage Guidelines

### Text Colors

```css
/* Primary text */
.text-primary {
  color: var(--color-black); /* #17171a */
}

/* Secondary text */
.text-secondary {
  color: var(--color-dark-gray); /* #26262a */
}

/* Tertiary text */
.text-tertiary {
  color: var(--color-gray); /* #51515c */
}

/* Placeholder text */
.text-placeholder {
  color: var(--color-light-gray); /* #9e9ea9 */
}
```

### Background Colors

```css
/* Primary background */
.bg-primary {
  background-color: var(--bg-white);
}

/* Card background */
.bg-card {
  background-color: var(--bg-light-gray); /* #f9f9f9 */
}

/* Navbar background */
.bg-navbar {
  background-color: var(--bg-gray); /* #f3f3f4 */
}
```

### Button Colors

```css
/* Primary button */
.button-primary {
  background-color: var(--color-black); /* #17171a */
  color: var(--bg-off-white); /* #fcfcfc */
}

/* Vote for button (active) */
.button-vote-for-active {
  background-color: var(--vote-for-active); /* #5861ff */
  color: var(--bg-white);
}

/* Vote against button (active) */
.button-vote-against-active {
  background-color: var(--vote-against-active); /* #ff5858 */
  color: var(--bg-white);
}
```

## Accessibility

### Contrast Ratios

All text color combinations meet WCAG AA standards (4.5:1 for normal text):

| Foreground | Background | Ratio | WCAG Level |
|------------|------------|-------|------------|
| `#17171a` (black) | `#ffffff` (white) | 19.8:1 | AAA |
| `#26262a` (dark gray) | `#ffffff` (white) | 16.3:1 | AAA |
| `#51515c` (gray) | `#ffffff` (white) | 7.8:1 | AAA |
| `#006044` (success) | `#d0fae4` (success bg) | 5.2:1 | AA |
| `#9e0711` (error) | `#fee1e1` (error bg) | 6.8:1 | AA |
| `#963b00` (warning) | `#fef3c6` (warning bg) | 5.5:1 | AA |

### Color Blindness

The Kopi color system is designed with color blindness in mind:

- Primary interactions don't rely solely on color
- Vote colors use distinct values (not just red/green)
- Semantic colors include text labels alongside color
- Sufficient contrast between all interactive states

## Best Practices

### Do's

✅ Use `#17171a` (black) for primary text and buttons
✅ Use `#f9f9f9` (light gray) for card backgrounds
✅ Use `#f3f3f4` (gray) for navbar backgrounds
✅ Use semantic colors for notifications (success, error, warning)
✅ Use vote colors only for voting interactions
✅ Maintain consistent background hierarchy (white → almost white → off white → light gray)

### Don'ts

❌ Don't use brand purple/orange for non-voting buttons
❌ Don't mix vote colors with other button types
❌ Don't use colors outside the defined palette
❌ Don't rely solely on color to convey meaning
❌ Don't use insufficient contrast ratios

## Component Color Usage

### Button Component

- **Primary**: Background `#17171a`, text `#fcfcfc`
- **Vote For (default)**: Background `#eeefff`, text `#17171a`
- **Vote For (active)**: Background `#5861ff`, text `#ffffff`
- **Vote Against (default)**: Background `#ffeeee`, text `#17171a`
- **Vote Against (active)**: Background `#ff5858`, text `#ffffff`
- **Vote Neutral (active)**: Background `#70707b`, text `#ffffff`

### Card Component

- **Background**: `#f9f9f9`
- **Text**: `#17171a` (primary), `#26262a` (secondary)

### Navbar Component

- **Background**: `#f3f3f4`
- **Active Item**: Custom styling per variant

### Notification Component

- **Success**: Background `#d0fae4`, text `#006044`
- **Error**: Background `#fee1e1`, text `#9e0711`
- **Warning**: Background `#fef3c6`, text `#963b00`
