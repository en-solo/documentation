## Overview

The Kopi typography system uses two primary typefaces to create clear hierarchy and excellent readability across all interfaces.

## Typefaces

### Primary: Poppins

Poppins is used for body text, UI elements, and general content. It provides excellent readability at all sizes.

- **Usage**: Body text, buttons, form labels, UI components
- **Weights**: 400 (Regular), 500 (Medium), 600 (Semi-bold), 700 (Bold)
- **Character Set**: Latin, Latin Extended

### Secondary: Readex Pro

Readex Pro is used for headings and emphasis, providing visual distinction and hierarchy.

- **Usage**: Headings, titles, display text
- **Weights**: 400 (Regular)
- **Character Set**: Latin, Latin Extended

## Usage Guidelines

### Hierarchy

Create clear visual hierarchy by using:
1. **Display/Large headings** for page titles
2. **Medium/Small headings** for section organization
3. **Body text** for main content
4. **Note text** for supplementary information
5. **Captions** for metadata and fine print

### Readability

- **Line Length**: Aim for 50-75 characters per line for optimal readability
- **Line Height**: Use the defined line heights; they're optimized for each size
- **Color Contrast**: Ensure WCAG AA compliance (4.5:1 minimum for body text)
- **Alignment**: Use left-aligned text for paragraphs; center alignment sparingly

### Accessibility

- Minimum font size for body text: 16px
- Minimum contrast ratio: 4.5:1 for body text, 3:1 for large text (18px+)
- Use semantic HTML (`<h1>`, `<h2>`, `<p>`, etc.)
- Avoid using color alone to convey information

## CSS Implementation

/* Font imports */
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;500;600;700&display=swap');
@import url('https://fonts.googleapis.com/css2?family=Readex+Pro:wght@400&display=swap');

  /* Font Families */
  --font-primary: 'Poppins', -apple-system, BlinkMacSystemFont, sans-serif;
  --font-heading: 'Readex Pro', -apple-system, BlinkMacSystemFont, sans-serif;

  /* Headings */
  --text-heading-medium-size: 24px;
  --text-heading-medium-height: 32px;
  --text-heading-medium-spacing: -1px;

  --text-heading-small-size: 20px;
  --text-heading-small-height: 28px;

  /* Body */

  --text-body-height: 24px;
  --text-body-spacing: 0px;

  /* Note */
  --text-note-size: 14px;
  --text-note-height: 20px;
  --text-note-spacing: 0px;

  /* Caption */
  --text-caption-size: 12px;
  --text-caption-height: 16px;
  --text-caption-spacing: 0px;
}

/* Heading Styles */
