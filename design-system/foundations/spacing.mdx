## Overview

The Kopi spacing system uses an 8px base unit for consistency across all components and layouts. This creates a harmonious rhythm and makes spacing decisions predictable and scalable.

## Spacing Scale

Our spacing scale is based on multiples of 8px, extracted from actual component implementations.

### Core Scale

| Token | Value | Rem | Usage | Example |
|-------|-------|-----|-------|---------|
| `space-1` | 8px | 0.5rem | Minimal spacing, icon gaps | Gap between icon and label in buttons |
| `space-2` | 16px | 1rem | Standard spacing, padding | Horizontal padding in buttons |
| `space-4` | 32px | 2rem | Section spacing | Spacing between UI sections |
| `space-8` | 64px | 4rem | Large spacing | Major layout sections |

### Component Spacing

Extracted from the Button component:

| Property | Value | Token | Usage |
|----------|-------|-------|-------|
| **Padding Y** (sm) | 8px | `space-1` | Vertical padding for small buttons |
| **Padding X** (sm) | 16px | `space-2` | Horizontal padding for small buttons |
| **Gap** | 8px | `space-1` | Space between icon and label |
| **Border Radius** | 16px | `radius-2` | Button corner radius |

## Usage Guidelines

### Padding

Use padding to create space within components:

```css
/* Small button */
padding: 8px 16px; /* space-1 space-2 */

/* Medium button */
padding: 12px 24px;

/* Large button */
padding: 16px 32px; /* space-2 space-4 */
```

### Gaps

Use gaps for spacing between elements:

```css
/* Icon and label */
gap: 8px; /* space-1 */

/* Form fields */
gap: 16px; /* space-2 */

/* Card grid */
gap: 24px; /* space-3 */
```

### Margins

Use margins for spacing between components:

```css
/* Tight spacing */
margin-bottom: 8px; /* space-1 */

/* Standard spacing */
margin-bottom: 16px; /* space-2 */

/* Section spacing */
margin-bottom: 32px; /* space-4 */
```

## Implementation

### CSS Variables

```css
:root {
  /* Base unit */
  --space-base: 8px;

  /* Spacing scale */
  --space-1: 8px;   /* 0.5rem */
  --space-2: 16px;  /* 1rem */
  --space-3: 24px;  /* 1.5rem */
  --space-4: 32px;  /* 2rem */
  --space-5: 40px;  /* 2.5rem */
  --space-6: 48px;  /* 3rem */
  --space-8: 64px;  /* 4rem */
  --space-10: 80px; /* 5rem */
  --space-12: 96px; /* 6rem */
  --space-16: 128px; /* 8rem */

  /* Component-specific */
  --btn-padding-y-sm: var(--space-1);
  --btn-padding-x-sm: var(--space-2);
  --btn-gap: var(--space-1);
}
```

## Layout Patterns

### Container Padding

```css
.container {
  padding-left: var(--space-4);  /* 32px */
  padding-right: var(--space-4); /* 32px */
}

@media (min-width: 768px) {
  .container {
    padding-left: var(--space-6);  /* 48px */
    padding-right: var(--space-6); /* 48px */
  }
}
```

### Stack Spacing

```css
.stack > * + * {
  margin-top: var(--space-2); /* 16px */
}

.stack-lg > * + * {
  margin-top: var(--space-4); /* 32px */
}
```

### Grid Gaps

```css
.grid {
  display: grid;
  gap: var(--space-2); /* 16px */
}

.grid-lg {
  gap: var(--space-4); /* 32px */
}
```

## Border Radius

The Kopi design system uses consistent border radius values:

| Token | Value | Usage |
|-------|-------|-------|
| `radius-sm` | 8px | Small elements, tags |
| `radius-md` | 12px | Cards, inputs |
| `radius-lg` | 16px | Buttons, modals |
| `radius-xl` | 20px | Large surfaces |
| `radius-full` | 9999px | Pills, circular elements |

**From Button component**: `border-radius: 16px` (radius-lg)

## Do's and Don'ts

### Do's

✅ Use the 8px spacing scale consistently
✅ Maintain rhythm by using multiples of 8px
✅ Use spacing tokens instead of hard-coded values
✅ Apply consistent spacing across similar components

### Don'ts

❌ Don't use arbitrary spacing values
❌ Don't mix different spacing systems
❌ Don't use spacing smaller than 8px except for optical adjustments
❌ Don't create visual tension with inconsistent spacing

## Responsive Spacing

Adjust spacing for different screen sizes:

```css
.section {
  padding: var(--space-4); /* 32px mobile */
}

@media (min-width: 768px) {
  .section {
    padding: var(--space-6); /* 48px tablet */
  }
}

@media (min-width: 1024px) {
  .section {
    padding: var(--space-8); /* 64px desktop */
  }
}
```

## Component Examples

### Button Spacing

```css
.button {
  padding: var(--btn-padding-y-sm) var(--btn-padding-x-sm); /* 8px 16px */
  gap: var(--btn-gap); /* 8px */
  border-radius: 16px;
}
```

### Card Spacing

```css
.card {
  padding: var(--space-4); /* 32px */
  border-radius: var(--radius-md); /* 12px */
}

.card-title {
  margin-bottom: var(--space-2); /* 16px */
}

.card-content {
  margin-bottom: var(--space-3); /* 24px */
}
```

}

  margin-top: var(--space-1); /* 8px */
