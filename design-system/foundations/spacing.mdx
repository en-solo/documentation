## Overview

The Kopi spacing system uses a refined scale from 2px to 48px for creating consistent layouts and component spacing. The system provides flexibility while maintaining rhythm and visual harmony.

## Spacing Scale

Our spacing scale provides 10 distinct values for all layout and component spacing needs.

### Core Scale

| Token | Value | Rem | Usage | Example |
|-------|-------|-----|-------|------------|
| `space-xxs` | 2px | 0.125rem | Minimal gaps, fine adjustments | Subtle borders, optical adjustments |
| `space-xs` | 4px | 0.25rem | Tiny spacing | Icon padding, tight gaps |
| `space-sm` | 6px | 0.375rem | Small spacing | Compact padding |
| `space-sm-plus` | 7px | 0.4375rem | Adjusted small spacing | Optical refinements |
| `space-md` | 8px | 0.5rem | **Base unit** | Icon gaps, button padding (vertical), component spacing |
| `space-lg` | 12px | 0.75rem | Medium spacing | Moderate padding and gaps |
| `space-xl` | 16px | 1rem | Standard spacing | Button padding (horizontal), component padding, card spacing |
| `space-2xl` | 20px | 1.25rem | Large spacing | Section padding, generous component spacing |
| `space-3xl` | 24px | 1.5rem | Extra large spacing | Major layout sections, navbar spacing |
| `space-4xl` | 48px | 3rem | Maximum spacing | Hero sections, major page divisions |

<Info>
The base unit is **8px** (`space-md`), used throughout components for consistent rhythm. Most component spacing is built around multiples of 8px.
</Info>

## Usage Guidelines

### Padding

Use padding to create space within components:

```css
/* Button - small */
padding: 8px 16px; /* space-md space-xl */

/* Card */
padding: 16px; /* space-xl */

/* Navbar */
padding: 24px; /* space-3xl */

/* Hero section */
padding: 48px; /* space-4xl */
```

### Gaps

Use gaps for spacing between elements:

```css
/* Icon and label (Button) */
gap: 8px; /* space-md */

/* Navbar items */
gap: 8px; /* space-md */

/* Form fields */
gap: 12px; /* space-lg */

/* Card grid */
gap: 16px; /* space-xl */

/* Major sections */
gap: 24px; /* space-3xl */
```

### Margins

Use margins for spacing between components:

```css
/* Minimal spacing */
margin-bottom: 8px; /* space-md */

/* Standard spacing */
margin-bottom: 16px; /* space-xl */

/* Section spacing */
margin-bottom: 24px; /* space-3xl */

/* Large spacing */
margin-bottom: 48px; /* space-4xl */
```

## Border Radius

The Kopi design system uses consistent border radius values:

| Token | Value | Usage |
|-------|-------|-------|
| `radius-xs` | 2px | Minimal rounding, subtle edges |
| `radius-sm` | 4px | Small elements, tight radius |
| `radius-md` | 5px | Medium radius |
| `radius-lg` | 8px | Moderate rounding |
| `radius-xl` | 16px | **Standard radius** - Buttons, inputs, cards, navbar items, notifications |
| `radius-2xl` | 20px | Large radius - Header component |

<Note>
**Most common radius**: 16px (`radius-xl`) is used for almost all Kopi components including buttons, inputs, cards, navbar items, and notifications.
</Note>

### Border Radius Implementation

```css
:root {
  /* Border radius scale */
  --radius-xs: 2px;
  --radius-sm: 4px;
  --radius-md: 5px;
  --radius-lg: 8px;
  --radius-xl: 16px;  /* Most commonly used */
  --radius-2xl: 20px;
  --radius-full: 9999px; /* Fully rounded */

  /* Component-specific */
  --btn-radius: var(--radius-xl);
  --card-radius: var(--radius-xl);
  --input-radius: var(--radius-xl);
  --navbar-item-radius: var(--radius-xl);
  --notification-radius: var(--radius-xl);
  --header-radius: var(--radius-2xl);
}
```

## Layout Patterns

### Container Padding

```css
.container {
  padding-left: var(--space-xl);  /* 16px */
  padding-right: var(--space-xl); /* 16px */
}

@media (min-width: 768px) {
  .container {
    padding-left: var(--space-3xl);  /* 24px */
    padding-right: var(--space-3xl); /* 24px */
  }
}

@media (min-width: 1024px) {
  .container {
    padding-left: var(--space-4xl);  /* 48px */
    padding-right: var(--space-4xl); /* 48px */
  }
}
```

## Do's and Don'ts

### Do's

✅ Use the defined spacing scale consistently
✅ Use `8px` (space-md) as the base unit for most component spacing
✅ Use `16px` (space-xl) for standard padding and gaps
✅ Use `24px` (space-3xl) for major sections
✅ Apply consistent spacing across similar components
✅ Use spacing tokens instead of hard-coded values

### Don'ts

❌ Don't use arbitrary spacing values outside the scale
❌ Don't mix different spacing systems
❌ Don't use spacing smaller than 2px except for sub-pixel rendering
❌ Don't create visual tension with inconsistent spacing
❌ Don't use excessive spacing that breaks visual relationships

## Responsive Spacing

Adjust spacing for different screen sizes:

```css
.section {
  padding: var(--space-xl); /* 16px mobile */
}

@media (min-width: 768px) {
  .section {
    padding: var(--space-3xl); /* 24px tablet */
  }
}

@media (min-width: 1024px) {
  .section {
    padding: var(--space-4xl); /* 48px desktop */
  }
}
```
