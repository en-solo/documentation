## Spacing Scale

Our spacing system is based on a **4px base unit** for consistency and mathematical precision across all designs.

### Base Scale

| Token | Value | Rem | Usage |
|-------|-------|-----|-------|
| `space-0` | 0px | 0rem | No spacing |
| `space-0.5` | 2px | 0.125rem | Extra fine spacing, optical alignment |
| `space-1` | 4px | 0.25rem | Minimal spacing, tight groups |
| `space-1.5` | 6px | 0.375rem | Fine spacing adjustments |
| `space-2` | 8px | 0.5rem | Small gaps, icon spacing |
| `space-2.5` | 10px | 0.625rem | Between small and compact |
| `space-3` | 12px | 0.75rem | Compact spacing |
| `space-4` | 16px | 1rem | Default spacing unit |
| `space-5` | 20px | 1.25rem | Medium spacing |
| `space-6` | 24px | 1.5rem | Comfortable spacing |
| `space-8` | 32px | 2rem | Large spacing |
| `space-10` | 40px | 2.5rem | Extra large spacing |
| `space-12` | 48px | 3rem | Section spacing |
| `space-16` | 64px | 4rem | Large section spacing |
| `space-20` | 80px | 5rem | Major section dividers |
| `space-24` | 96px | 6rem | Page-level spacing |

<Tip>
  Our 4px-based grid system provides more flexibility than traditional 8px systems. Use multiples of 4px (space-1, space-2, space-3, etc.) for precise control over spacing.
</Tip>

### External Margins

Spacing between components and sections:

| Context | Spacing | Token |
|---------|---------|-------|
| Between related elements | 8px - 12px | `space-2` to `space-3` |
| Between form fields | 16px - 20px | `space-4` to `space-5` |
| Between cards in a grid | 16px - 24px | `space-4` to `space-6` |
| Between sections | 32px - 48px | `space-8` to `space-12` |
| Between major page sections | 64px - 96px | `space-16` to `space-24` |

## Layout Grid

### Container Widths

| Breakpoint | Container Max Width | Padding |

## Vertical Rhythm

Maintain consistent vertical spacing throughout your layouts using the spacing scale.

### Typography Spacing

| Element | Top Margin | Bottom Margin |
|---------|------------|---------------|
| H1 | 0 | 24px (`space-6`) |
| H2 | 40px (`space-10`) | 16px (`space-4`) |
| H3 | 32px (`space-8`) | 12px (`space-3`) |
| H4 | 24px (`space-6`) | 12px (`space-3`) |
| Paragraph | 0 | 16px (`space-4`) |
| List | 0 | 16px (`space-4`) |
| List items | 0 | 8px (`space-2`) |

### Content Sections

<Steps>
  <Step title="Related content groups">
    Use 8px - 16px (`space-2` to `space-4`) between tightly related items
  </Step>

  <Step title="Content blocks">
    Use 24px - 32px (`space-6` to `space-8`) between different content blocks
  </Step>

  <Step title="Major sections">
    Use 48px - 64px (`space-12` to `space-16`) between major page sections
  </Step>

  <Step title="Page divisions">
    Use 80px - 96px (`space-20` to `space-24`) between distinct page areas
  </Step>
</Steps>

## Responsive Spacing

Adjust spacing based on screen size to optimize for different devices:

### Mobile First Approach

```css
/* Mobile (default) */
.section {
  padding: 32px 16px; /* space-8 space-4 */
  gap: 16px; /* space-4 */
}

/* Tablet and up */
@media (min-width: 768px) {
  .section {
    padding: 48px 24px; /* space-12 space-6 */
    gap: 24px; /* space-6 */
  }
}

/* Desktop and up */
@media (min-width: 1024px) {
  .section {
    padding: 64px 32px; /* space-16 space-8 */
    gap: 32px; /* space-8 */
  }
}
```

## Stack & Inline Spacing

### Stack (Vertical)

For vertical spacing between stacked elements:

```css
.stack {
  display: flex;
  flex-direction: column;
  gap: 16px; /* space-4 - adjust as needed */
}
```

### Inline (Horizontal)

For horizontal spacing between inline elements:

```css
.inline {
  display: flex;
  flex-direction: row;
  gap: 8px; /* space-2 - adjust as needed */
}
```

## Usage Guidelines

<CardGroup cols={2}>
  <Card title="Use the Scale" icon="ruler">
    Always use values from the spacing scale. Avoid arbitrary values like 15px or 23px.
  </Card>

  <Card title="Think in Multiples" icon="calculator">
    Stick to multiples of 4px using our spacing scale. Half-values (0.5, 1.5, 2.5) provide fine-tuning options.
  </Card>

  <Card title="Group Related Items" icon="object-group">
    Use smaller spacing (8px-16px) for related items, larger spacing (32px+) for separate sections.
  </Card>

  <Card title="Maintain Consistency" icon="equals">
    Use the same spacing values for similar contexts throughout your design.
  </Card>
</CardGroup>

## Accessibility Considerations

<Warning>
  Ensure touch targets are at least 44px × 44px for mobile devices, including the spacing around interactive elements.
</Warning>

### Minimum Spacing

- **Touch targets**: 44px minimum (including padding/margin)
- **Text line spacing**: 1.5× font size minimum for body text
- **Interactive elements**: 8px minimum spacing between clickable items

## Common Patterns

### Form Layouts

```css
/* Form field group */
.form-group {
  margin-bottom: 20px; /* space-5 */
}

/* Label to input */
.label {
  margin-bottom: 8px; /* space-2 */
}

/* Helper text */
.helper-text {
  margin-top: 8px; /* space-2 */
}

/* Form sections */
.form-section {
  margin-bottom: 40px; /* space-10 */
}
```

### Card Grids

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 24px; /* space-6 */
  padding: 32px; /* space-8 */
}
```

<Info>
  When implementing spacing, use CSS custom properties or design tokens to ensure consistency and make global updates easier.
</Info>
