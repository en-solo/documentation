## Design Principles

<AccordionGroup>
  <Accordion title="Consistency" icon="equals">
    Maintain visual and functional consistency across all touchpoints to create a cohesive user experience.
  </Accordion>

  <Accordion title="Accessibility" icon="universal-access">
    Design for all users, ensuring WCAG 2.1 AA compliance at minimum for color contrast, keyboard navigation, and screen readers.
  </Accordion>

  <Accordion title="Scalability" icon="arrow-up-right-dots">
    Build flexible components and patterns that adapt to different contexts and scale across products.
  </Accordion>

  <Accordion title="Performance" icon="gauge-high">
    Optimize for speed and efficiency in both design and implementation.
  </Accordion>
</AccordionGroup>
